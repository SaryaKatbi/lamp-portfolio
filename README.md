## Run Locally

Clone the project

```bash
  git clone https://github.com/SaryaKatbi/lamp-portfolio.git
```

Go to the project directory

```bash
  cd lamp-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run
```
